# RemixFig: 中文提示词

需要一个支持 GPT-5.5 访问权限的 ChatGPT Plus 账号。在ChatGPT新对话中上传你的框架图，然后依次发送以下 5 条消息。每条消息等GPT完整回复后再发下一条。


**第 1 步**

```
分析这张图，把所有元素（框、文字、箭头、图标）的位置和属性输出成JSON，坐标用百分比表示，不要生成任何图或代码
```


**第 2 步**

```
按照你上面分析出来的这些坐标的位置，重新画一个每个元素可编辑的ppt图，不规则的元素保留元素本身作为整体
```


**第 3 步**

```
把这个图所有的图标元素都生成一模一样清晰的版本排列到一个空白图里。每个图标在原始图里面对应的坐标也输出出来json文件
```


**第 4 步**

```
把上述每个图标在原始图里面的位置和属性输出成JSON，坐标用百分比表示，不要生成任何图或代码
```


**第 5 步**

```
现在把这些图标根据上述精确位置替换生成的ppt的对应的图标，再生成一版可编辑的ppt图
```
